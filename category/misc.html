vim 中判断在终端还是在gui运行

2019-02-21 星期四 14:09 | -- (permalink)

判断vim终端还是在gui运行

遇到这样一个问题，在终端下运行vim，配色不太好看，如果直接在vimrc里改，则gvim也受到影响。 不过打开我的vimrc文件看了一下，是有现成答案的。

if has('gui_running')
    if strftime("%H")>6    
        colorscheme  darkblue
    endif
    if strftime("%H")>22       
        colorscheme  maroloccio
    endif
    if strftime("%H")<6
        colorscheme  maroloccio
    endif
else
  colorscheme darkburn
endif

用这样一个判断就可以了。 如何简单。
