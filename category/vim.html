  2. vim中的autocmd

    2016-01-19 星期二 08:58
    By 天地一沙鸥

    原来做了个vim的写日记的模板，并且为它写了语法高亮文件。效果还是可以的，不过有个问题，就是，写新的日记的时候，不能语法高亮，只能保存后才能高亮，也试过autocmd的其它形式，也是不行，不过，昨天在尝试autocmd的时候，无意中发发现了解决办法。

    autocmd BufReadpre *.mytext set syn=mytext
mytext是指代自己定义的文件格式和语法文件

  3. vim中的find命令，打开文件很方便

    2016-01-08 星期五 15:07
    By 天地一沙鸥

    这几天看《Vim实用技巧》,其中介绍了 :find 命令的使用

    在vim的配件文件.vimrc or _vimrc进行如下设置

    set path+=yourdir1/**,youdir2/**

    就可以在vim中方便的打开设置好的目录下的文件， 《Vim实用技巧》 还有更多的技巧，可以值得去看看。

    另外，在网上无意中发现《Vim实用技巧》的作者的博客易水博客,值得一看。
