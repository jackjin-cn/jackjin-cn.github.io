用find 查找满足多个条件的文件

网上看了教程，find 可以添加 -o 作为或者条件，我试了一下，-a 表示 and,居然也是可以的

下面就是找出文件名同时满足三个条件的用法，其它更高级的用法可以自行搜索

find . -name '关键词' -a -name '关键词2' -a -name '关键词'

加*作用是通配符，可以自行尝试。

vim中的find命令，打开文件很方便
by 天地一沙鸥 file under vim with tags vim, 文件, find, path

这几天看《Vim实用技巧》,其中介绍了 :find 命令的使用

在vim的配件文件.vimrc or _vimrc进行如下设置

set path+=yourdir1/**,youdir2/**

就可以在vim中方便的打开设置好的目录下的文件， 《Vim实用技巧》 还有更多的技巧，可以值得去看看。

另外，在网上无意中发现《Vim实用技巧》的作者的博客易水博客,值得一看。
