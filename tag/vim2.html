vim中的find命令，打开文件很方便

Permalink: 2016-01-08 15:07:00+08:00 by 天地一沙鸥 in vim tags: vim 文件 find path

这几天看《Vim实用技巧》,其中介绍了 :find 命令的使用

在vim的配件文件.vimrc or _vimrc进行如下设置

set path+=yourdir1/**,youdir2/**

就可以在vim中方便的打开设置好的目录下的文件， 《Vim实用技巧》 还有更多的技巧，可以值得去看看。

pandoc的用法（1）

Permalink: 2016-01-05 17:28:00+08:00 by 天地一沙鸥 in 软件 tags: pandoc vim html css

pandoc的用法（1）

用vim写markdown，自己在.vimrc里也写快捷命令进行预览，感觉还是很方便的。

pandoc在生成html文件的时候，可以支持css，还可以生成单独的文件，包含在图片里面。

pandoc -c style.css  #包含css样式（可以包括路径,但html和css是分离）
pandoc -s -with--contained  #生成单一文件（可以把css，图片的包含在一个html文件中,生成docx文件也可以用这个参数，图片被包含在docx文件中）
