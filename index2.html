git 本地删除后的问题

git 本地删除后的问题

在用pelican生成网页的时假，如果不小心用make clean的话，会把.git删除，就会产生配对问题，需要重新配对。

对于网页不需要保留远程文件，所以就如下操作

git init
git add .
git commit -m 'update'
git remote rm origin
git push --force origin master  强制推送
git push -u origin master 重新远程与本地对应

vim中的find命令，打开文件很方便

这几天看《Vim实用技巧》,其中介绍了 :find 命令的使用

在vim的配件文件.vimrc or _vimrc进行如下设置

set path+=yourdir1/**,youdir2/**

就可以在vim中方便的打开设置好的目录下的文件， 《Vim实用技巧》 还有更多的技巧，可以值得去看看。

另外，在网上无意中发现《Vim实用技巧》的作者的博客易水博客,值得一看。
