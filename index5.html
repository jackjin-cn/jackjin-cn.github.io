用find 查找满足多个条件的文件

网上看了教程，find 可以添加 -o 作为或者条件，我试了一下，-a 表示 and,居然也是可以的

更正，昨天看了一个教程，原来find的条件默认是and这样的关系。

下面就是找出文件名同时满足三个条件的用法，其它更高级的用法可以自行搜索

find . -name '关键词' -a -name '关键词2' -a -name '关键词'

加*作用是通配符，可以自行尝试。

Date 2018-07-11 星期三 09:40 By 天地一沙鸥 Category 软件. Tags python brew office command line

doc转docx

网上找了很多办法，想在mac把doc转成docx。一般有2种方案。

  1. antiword 可以把doc转成纯文件，或者docbook格式（这个我没有搞明白）
  2. 用unoconv + libreoffice.

昨天晚上试了一下第二种方案，libreoffice的版本要3.x和4.x，官网上只有5和6了，找了一下，还是找到了。

libreoffice 4下载,希望以后还能用。

安装好以后就可以使用了。

uniconv -f docx source.doc

这样就可以转换了，如果成功的话会生成一个同名的docx文件，当然，仔细看说明，应该也还可以指定输出文件名了。
